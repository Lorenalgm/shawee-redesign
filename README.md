### :wrench: Como Utilizar

1. Fork ou clone esse repositório:
```sh 
git@github.com:Lorenalgm/shawee-redesign.git
```
2. Abra o projeto no seu editor de texto
3. Instale os pacotes e inicie o server!
```sh 
    yarn
    yarn start
```
